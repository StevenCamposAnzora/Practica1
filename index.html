Semantica con HTML

El HTML semántico consiste en utilizar etiquetas que describen claramente el significado y propósito del contenido, tanto para humanos como para máquinas. A diferencia de elementos no semánticos como div o span, que no aportan contexto, los elementos semánticos como header, article o footer transmiten estructura y función del contenido.
